取血`1滴`置载玻片一端，以边缘平滑的推片，从血滴前方接触血液，使血液沿推片散开，通常推片与载玻片保持`25°～30°`夹角，平稳地向前推动，血液即在载玻片上形成薄层血膜。

```js
1. 涂片厚薄与血滴大小、推片与载玻片间夹角、推片速度、血细胞比容有关。

2. 一张良好的血片，应厚薄适宜、头体尾明显、细胞分布均匀、血膜边缘整齐，并留有一定空隙。

干扰项：`❌血膜长度占载玻片长度的 3/4 左右` // [!code error]

```
