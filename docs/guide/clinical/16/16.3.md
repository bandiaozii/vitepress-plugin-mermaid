::::tip

```js
1. Ⅰ、Ⅱ 度:
    为正常。
2. Ⅲ、Ⅳ 度:
    为不清洁，常可同时发现病原微生物，提示存在感染引起的阴道炎。
3. 阴道清洁度与卵巢功能有关:
    排卵前期阴道趋于清洁。雌激素减低阴道不清洁。
```
