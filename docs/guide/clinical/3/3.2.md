## (2)方法学评价

<son :text="'临床检验基础检验记忆卡'" text1="(2)方法学评价" :textOption="[['了解','相关专业知识','专业实践能力'],['了解','专业知识','专业实践能力'],['了解','专业知识','专业实践能力']]" />

::::tip

```js
1. 显微镜计数法
   简便易行，但重复性和准确性较差。
2. 血液分析仪计数法
   计数细胞数量多，速度快，易于标准化，计数精确性较高，但需特殊仪器。
   某些人为因素（如抗凝不充分）、病理情况（如出现有核红细胞、巨大血小板、血小板凝集等）可干扰白细胞计数。
   使用前须按 NCCLS 规定方法对仪器进行校准，且须认真坚持日常质控工作。

```

::::
